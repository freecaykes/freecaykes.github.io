libselfie

An Android library for camera applications to capture the frame with least changes between previous and successive frames.

Using OpenCV's background subtractor, the library analyzes a span of successivve frames and extracts the one with the least tranistional changes from one frame to the next using an evolutionary algorithm to calculate the frame with the lowest derivative of noise changes.
